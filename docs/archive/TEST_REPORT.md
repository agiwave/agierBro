## 问题修复

### 已修复问题

1. **类型错误** - ToolButtons 使用了不存在的 `activeData` 属性
   - 修复：改用 `currentData`
   
2. **类型错误** - TreeNode 引用了未定义的 `activeNode`
   - 修复：移除引用

3. **URL 路径问题** - 内容 URL 没有 `/api/` 前缀
   - 修复：自动添加 `/api/` 前缀

4. **Schema 识别** - 不支持内置 Schema 字符串
   - 修复：添加 `@` 前缀识别
